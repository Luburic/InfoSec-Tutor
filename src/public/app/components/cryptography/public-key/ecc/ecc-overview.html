In this section I am going to teach you about Elliptic Curve Cryptography (ECC). Today we can find elliptic curve cryptosystems in SSH, PGP, TLS, Bitcoins and other protocols, cryptocurrencies and technologies.

This section, like most others in the CRYPTOGRAPHY group, consists of the Algorithm subsection, where I'll focus primarily on the actual steps of the algorithms, after which I'll draw some comparisons with the RSA algorithms and show you some real world applications of ECC.

However, before you jump into the algorithm you should probably check out the Elliptic Curves subsection where I explain the elliptic curves in detail, focusing on cyclic subgroups of elliptic curves over finite fields and any additional constraints needed for ECC. Don't worry, I'll include some reminders about algebraic groups and fields so you should be able to understand everything with little effort. I should give you a fair warning, you probably definitely won't understand the algorithms without understanding the elliptic curves themselves.

The next subsection, titled Demo, will allow you to play with both elliptic curves and the algorithms based on said curves. You can configure the parameters of the algorithm and see some output and metrics for custom input. Here we also include some benchmark results of the algorithm we used for various sets of input.

Last, but certainly not least, we provide you with several links and references in the What Next? subsection, which should further develop your knowledge on the subject.
