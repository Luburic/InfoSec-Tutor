This section is all about Rivest-Shamir-Adleman Algorithm (RSA). These days you can find RSA in many commercial systems. It is used by web servers and browsers to secure web traffic (e.g. in the SSL), it is used to ensure privacy and authenticity of e-mail (e.g. in PGP), it is used to guarantee remote connection in SSH, and it has an important role in the modern payment systems through SET protocol. In short, RSA is frequently used in applications where security of digital data is a concern.
